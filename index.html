Giới thiệu về SCHOOL WEB

Ngày 10, tháng 1, năm 2020

School Web là một website chia sẽ kiến thức về lập trình web. Nếu bạn chưa biết gì về lập trình web, thì trang web này là giành cho bạn. School Web bao gồm những bài học căn bản Front-End và Back-End được sắp xếp theo thứ tự, gom nhóm theo từng ngôn ngữ, kỹ thuật cụ thể đi kèm với ví dụ thực tế và hướng dẫn từng bước một giúp bạn hiểu rõ, sâu về vấn đề mình đã học. School Web cũng là một trang web có nội dung miễn phí, do đó bạn có thể sử dụng nội dung một cách rất tự nhiên mà không cần phải lo ngại về mặt bản quyền.


Nội dung School Web

School Web bao gồm các bài học về các ngôn ngữ và lĩnh vực sau đây:

  1. HTML
  2. CSS
  3. JavaSCript
  4. PHP
  5. C#
  6. Node.js
  7. SQL
  8. Front-End
  9. Back-End

Trong mỗi phần tôi sẽ thảo luận một cách chi tiết và đưa ra viết dụ kèm theo để bạn có thể hiểu rõ, sâu về vấn đề mình đang học.
